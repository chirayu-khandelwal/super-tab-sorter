Welcome to Super Tab Sorter!

Please review the User Guide before getting started.

You can right-click on the extension icon and select "Options" at any time to configure extension settings and review the user guide.

Super Tab Sorter has been updated to v!

This version uses the new Chrome manifest v3 API (major architectural change) and adds direct support for Chrome's tab groups feature.

It also adds optional duplicate tab removal (during sorting) and Sort By URL sorts by root domain and sub-sorts by host name.

Please review the updated User Guide to learn more about the latest changes.
